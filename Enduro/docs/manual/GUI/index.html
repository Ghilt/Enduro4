Förberedelser

För att kunna köra programmet krävs det att din dator har Java. Java finns att ladda ner här:

Java

Programmet kan sammanställa filer av följande slag:

  • Fil med startnummer
  • Fil med måltider
  • Fil med namn på deltagare

Alla filer ska inkludera förarens startnummer och dessa filer ska anges vid start av programmet.

Installation

Nedan följer instruktioner om hur programmet installeras.

  • Anslut USB-minnet med programmet till datorn.
  • Gå till "Den här datorn" och öppna USB-minnet.
  • Gå till mappen "modmqwo".
  • Starta JAR-filen "namn". Programmet är nu reda att användas!

Beskrivning
